# Code Optimization Visualizer

A mini project for SPCC (System Programming and Compiler Construction).

This project demonstrates compiler optimization techniques using Python and Streamlit.

## Features

- Constant Folding
- Common Subexpression Elimination
- Dead Code Elimination
- Step-by-step optimization visualization

## Technologies Used

- Python
- Streamlit
- Regular Expressions

## Run Locally

```bash
pip install streamlit
streamlit run app.py
